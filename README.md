# gdi_TicTacToe
Das vorliegende WindowsForms-Projekt soll TicTacToe mit GDI+ umsetzen.


## :dart: Ziele
- Objekte in mehrdimensinalen Arrays verwalten.
- Fachlichen Zustand (Gamestate) von Darstellung getrennt verwalten.
